# آرمک تسک (Armec Task)

اپلیکیشن مدیریت تسک‌های روزانه و هفتگی با رابط کاربری مینیمال و رنگ‌های پاستیلی.
ساخته‌شده با **React Native + Expo**.

## ویژگی‌ها

- ✅ افزودن / ویرایش / حذف تسک
- ⏰ تعریف ددلاین (تاریخ و ساعت) و آلارم در زمان ددلاین
- 🔁 تسک‌های تکراری: هر روز / هر هفته / هر چند روز (مثلاً هر ۳ روز)
- 🔔 نوتیفیکیشن تسکی که **نزدیک‌ترین ددلاین** را دارد
- 😴 قابلیت **اسنوز** (۱۰ دقیقه)
- 🏷 دسته‌بندی با رنگ دلخواه (افزودن/ویرایش/حذف)
- ⭐ اولویت‌بندی: زیاد / متوسط / کم
- 📅 تقویم **شمسی** ماه با نمایش تسک‌ها و موضوع محتوا
- 📊 صفحه گزارش: کارهای انجام‌شده به تفکیک دسته + **خروجی اکسل**
- 💾 ذخیره‌سازی **محلی** روی گوشی (بدون سرور)
- 📤 انتقال بین گوشی‌ها: **خروجی/ورودی بکاپ (JSON)**

## نصب و اجرا (توسعه‌دهنده)

```bash
npm install --legacy-peer-deps
npx expo start
```

برای اجرا روی گوشی: اپ **Expo Go** را نصب کنید و QR کد را اسکن کنید.

## ساخت APK

از طریق EAS (نیاز به اکانت رایگان expo.dev):

```bash
npm install -g eas-cli
eas login
eas build -p android --profile preview   # خروجی APK
# یا برای نسخه نهایی:
eas build -p android --profile production
```

فایل APK ساخته‌شده را دانلود و روی گوشی نصب کنید.

## ساختار پروژه

```
src/
  storage.js        مدیریت دیتا (AsyncStorage)
  notifications.js  آلارم و نوتیفیکیشن
  calendar.js       توابع تقویم شمسی
  theme.js          رنگ‌های پاستیلی
  screens/
    HomeScreen.js       داشبورد / نزدیک‌ترین ددلاین
    TasksScreen.js      لیست و فرم تسک‌ها
    CalendarScreen.js   تقویم شمسی
    CategoriesScreen.js مدیریت دسته‌ها
    ReportScreen.js     گزارش + خروجی اکسل + بکاپ
```

## نکات

- تمام داده‌ها روی خود گوشی ذخیره می‌شوند.
- برای انتقال به گوشی دیگر: صفحه «گزارش» → خروجی بکاپ → در گوشی مقصد «وارد کردن بکاپ».
- برای آلارم در زمان دقیق، دسترسی «اعلان‌ها» را در اولین اجرا مجاز کنید.
